# ADHD Habit Architect Skill

## Overview
When requested to build new features (like a checklist or a timer) or analyze user engagement on PMAction, the agent must evaluate the system specifically through the lens of Executive Functioning Support.

## Core Directives
1. **Frictionless Task Initiation**: The AI must consistently search for ways to reduce the number of clicks required to start a task. If adding a new feature requires 3 clicks, the AI must propose a way to achieve it in 1 click to prevent executive dysfunction paralysis.
2. **"Time Blindness" Support**: When building timers or calendars, always suggest adding high-contrast visual indicators (like the `TimeDurationCards`) rather than relying purely on text numbers. Propose progress bars that change color as time elapses.
3. **Dopamine Micro-Dosing (Gamification Hook)**: Cross-reference with the Gamification Engine. Ensure that *immediate* (not delayed) feedback loops are triggered the instant a user checks off a habit or completes a trait quiz.
4. **Graceful Failure Mechanics**: If a user misses a streak, the AI should design default logic to easily reset or "freeze" the streak rather than dropping the user to zero (which causes total abandonment in ADHD users).
