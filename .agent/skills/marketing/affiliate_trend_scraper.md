# Affiliate & Trend Scraper Skill

## Overview
This skill is designed to turn the application or content site into a dynamic revenue generator. It uses web search integrations to find real-time trends related to the site's niche and connects those trends to monetizable actions.

## Duties
1. **Trend Analysis**: Utilize the `brave-search` MCP to query real-time news, trending products, or high-volume keywords related to the platform's core demographic.
2. **Product Matching**: Cross-reference trending topics with popular affiliate networks (e.g., Amazon Associates, ShareASale) or internal eCommerce offerings.
3. **Dynamic Content Injection**: 
   - Suggest code updates to dynamically pull in related products at the bottom of blog posts or specific app pages.
   - Propose UI components (like a "Trending Gear" or "Recommended Tools" widget) that seamlessly blend with the site's design.
4. **Link Management**: Instruct the agent on how to update a centralized database table (e.g., `affiliate_links`) to easily rotate trending products without needing hard-coded deployments.
5. **SEO & Context**: Ensure that inserted links provide genuine value and are contextually relevant to the surrounding content to maintain high SEO standing and user trust.
