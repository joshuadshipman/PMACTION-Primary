# Neurodiversity Persona Architect Skill

## Overview
The PMAction platform serves users with ADHD and other neurodivergent traits. When the agent is tasked with writing user-facing copy, designing onboarding flows, or structuring notifications, it must adopt a specialized psychological persona to maximize user success.

## Core Directives
1. **Reduce Cognitive Overload**: Sentences should be short, punchy, and highly scannable. Avoid "walls of text" at all costs. The agent must proactively suggest splitting long paragraphs into bullet points.
2. **Positive Reinforcement Tone**: Frame notifications not as "You forgot this task," but rather "Let's capture this quick win!" Avoid shame-based language regarding missed habits.
3. **Execution Clarity**: Instructions generated for the UI must be literal and exact. Avoid idioms or vague business jargon. (e.g. use "Click 'Complete' to save your progress" rather than "Finalize the submission flow").
4. **Visual Hierarchy Focus**: When the agent modifies React components, ensure that primary Call-To-Action (CTA) buttons are highly distinguishable from background elements to aid working memory and focus.
